About This Project

This project uses Azure Open Datasets, Azure Blob Storage, Microsoft Fabric, Lakehouse, Notebook transformations, Dataflow Gen2, and Power BI to build a large-scale NYC taxi analytics workflow. The exported PDF summary is included so the reporting output remains available even if a Fabric or Power BI trial expires.
